# WIN：

### 安装 WSL：

1. 检查CPU虚拟化是否开启：任务管理器 -> 性能 -> CPU -> 下方文字。

   若未启用，重启更改bios设置。（不同处理器略有不同，根据个人情况查硬件对应"bios 虚拟化"教程）

   <img src="C:\Users\a7743\AppData\Roaming\Typora\typora-user-images\image-20260326162459457.png" alt="image-20260326162459457" style="zoom:75%;" />

2. 安装WSL：

   在线安装（需梯子挂全局或者TUN模式）：

   ​	打开管理员终端，输入 `wsl --install` 安装wsl（默认安装到C盘且无法改路径，如有需求自行迁移到其他盘）

   离线安装：

   ​	搜索 “启用或关闭windows功能” -> 打开 “适用于 Linux 的 Windows 子系统” 、 “虚拟机平台”

   ​	重启电脑

   ​	复制资料中 wsl-ubuntu24.04.tar 绝对路径（右键，属性 位置）

   ​	打开管理员终端运行 `wsl --import Ubuntu-24.04 安装目录路径(建议不要在C盘) 刚刚复制的路径`

3. 打开终端，运行 `wsl` 检查是否安装成功

### 安装必要依赖(可选)：

1. __(可选)__  推荐一个终端工具 [Warp: The Agentic Development Environment](https://www.warp.dev/)

2. 打开终端，`wsl` 打开wsl

3. ```bash
   # 安装git, npm
   
   sudo apt update # 更新本地索引
   
   sudo apt install git -y # 安装git
   # 安装nvm(nodejs版本管理工具)
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash
   source ~/.bashrc # 重新加载配置
   nvm install --lts # 使用nvm安装nodejs长期支持版
   node --version # 查看node版本，检验是否成功安装
   ```
